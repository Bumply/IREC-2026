## Technical Report Outline (25 pages max for COTS)

### Main Body
1. **Executive Summary** (1 page)
2. **Introduction** (2 pages)
3. **System Architecture Overview**
   - Aero-structures
   - Propulsion
   - Avionics
   - Recovery
   - Telemetry
   - Payload
4. **Mission CONOPS**
5. **Conclusions & Lessons Learned**
6. **Acknowledgments**

### Appendices (no page limit)
- **A:** System Weights, Measures, & Performance Data
- **B:** Project Test Reports
- **C:** Hazard Analysis
- **D:** Risk Assessment
- **E:** Checklists
- **F:** Engineering Drawings
