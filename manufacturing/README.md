## Composite Process Steps

### Fiberglass Body Tubes
1. Clean 154mm aluminum mandrel
2. Apply release wax
3. Cut S-Glass fiberglass to size (with tolerance)
4. Apply epoxy resin
5. Wrap fiberglass layers
6. Vacuum bag
7. Oven cure (follow resin specs)
8. Trim excess
9. Polish and paint
10. **Final thickness: 2mm**

### Carbon Fiber Fins
1. Clean glass surface
2. Apply release agent
3. Cut carbon fiber sheets
4. Wet layup with epoxy
5. Stack layers for 7.5mm thickness
6. Vacuum bag
7. Oven cure
8. Trim to trapezoidal shape
9. Finish edges

### Carbon Fiber Nose Cone
1. Fabricate MDF mold (Von Kármán profile)
2. Apply release agent
3. Wet layup carbon fiber
4. Shape over mold
5. Vacuum bag
6. Oven cure
7. Trim and finish
8. Install aluminum tip
